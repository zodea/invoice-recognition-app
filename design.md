> Vue 3 + Tauri 桌面应用。送货单扫描件整理、发票识别校对、批量打印排版、分供方资料管理、材料单价对比。
> 本文是 stitch 出图的唯一输入，也是前端实现的基准参考。

## § 0 硬规则

1. **颜色唯一来源**：`src/styles.css :root` + `uno.config.ts theme.colors`。任何外部设计稿色值若与本文冲突，以本文为准。

## § 1 设计身份

### 1.1 产品定位

"工地记账员的桌面"——一切为吞吐量服务的本地工具。视觉气质接近银行柜台终端，不是 SaaS 仪表盘、不是营销官网。

### 1.2 视觉签名

本应用有两个区别于其他同类工具的视觉标记：

1. **中文单字图标方块**：导航项使用单个汉字（票/单/方/价）置于 `rounded-lg` 方块内，选中态白底蓝字、未选中态浅灰底。这是品牌辨识度的来源——替代 emoji 后，导航方块仍保留中文单字，不换成纯 Lucide 图标。
2. **A4 打印排版预览**：发票校对中栏不是普通文档列表，而是真实 A4 比例的打印排版网格，每页 1/2/4 张。用户看到的就是打印出来的样子。这是核心交互记忆点。

### 1.3 视觉密度

高密度、可扫读。正文 14px，label 11-12px font-600，标题 font-800 加粗。金融数据（发票号/税号/日期/金额）使用 `font-mono` tabular-nums。间距紧凑但不拥挤——panel 内 p-3~p-4，元素间 gap-2~gap-3。

### 1.4 色彩策略

```
画布 #f5f6f8  ──  机构灰，信号："这是严肃的财务工具"
面板 #ffffff  ──  白纸浮在灰桌面上，层级天然分明
品牌 #2563eb  ──  唯一非状态色，标记"你在哪"和"主操作"
状态三色     ──  红/橙/绿交通灯体系，每色三档（实色/浅底/深字）
```

不使用渐变、玻璃态、深色模式、装饰性色带。

### 3.5 统一交互模式

**sticky 操作栏**：所有视图的操作栏统一 `sticky top-16`。不允许操作入口被滚走。

**回到顶部**：`fixed bottom-6 right-6`，滚动超一屏淡入。圆形按钮 `i-lucide-arrow-up`。

**vxe-table**：所有表格统一 vxe-table。拖拽列宽(resizable)、固定首尾列、固定表头、hover 高亮。

**操作列图标化**：纯图标按钮 28×28 或 32×32，图标 16px，hover 显示 tooltip。danger 用 `text-danger`。

#### 左栏详细

- sticky 头部：批次统计 + 状态筛选条（全部/待人工校对/识别完成）
- 列表项：缩略图(7×9) + 单据编号 font-700 + 购买方/文件名 `text-ink-soft` + 状态 chip
- 选中态：`bg-brand-soft/60`
- 点击→中栏预览聚焦 + 右栏加载

#### 中栏详细

- sticky 头部："打印排版" + **下拉按钮**（"2张/页 ▾"），选项 1/2/4
- A4 页面：`aspect-ratio: 1 / 1.4142`，白底 + `border-line` + `shadow-card`
- 打印位：
  - 左上序号 chip（`bg-ink text-white`），如 `01` 或 `01-2`（多页）
  - 右上"待复核" chip（`bg-warn-soft text-warn-ink`）
  - 选中态：`border-brand shadow-[0_0_0_2px_rgba(37,99,235,.22)]`
- 交互：单击选中 / 双击放大 / 右键菜单（放大/左旋/右旋）
- **旋转粒度：按页**。多页 PDF 每页独立 rotation，旋转不影响同一发票的其他页

## § 10 动效

### 10.1 参数

时长 120~180ms（大弹窗 220ms），曲线 `cubic-bezier(0.2, 0, 0.2, 1)`，支持 `prefers-reduced-motion`。

### 10.2 五类动效

1. **阶段切换**：淡入 + 6~8px 纵向位移（`view-shift`）
2. **左右联动**：点击打印位→右栏对应行 scrollIntoView + 短暂高亮（`relation-pulse`）
3. **弹窗/抽屉**：遮罩淡入 + 内容上移（`dialog-pop`）
4. **树/卡片**：展开/关闭过渡 + 列表项进入（`tree-reveal` / `list-soft`）
5. **toast**：进入/退出动画；chip 状态变化短暂闪底色

### 10.3 禁止

全局路由大动画 / 表格行持续动画 / 金额滚动计数 / 拖拽物理弹簧

## § 11 状态规范

| 状态 | 表现 |
|---|---|
| 空状态 | 说明下一步操作（不留空白） |
| 读取中 | 行级 loading / skeleton |
| 处理中 | 按钮禁用 + 短文本 |
| 识别失败 | 行内错误 + 重试入口 |
| 部分失败 | 保留已识别内容，标记待复核 |
| 无匹配搜索 | "没有匹配"+ 清空筛选提示 |
| 能力缺失 | 提示改用桌面应用或下载 |

## § 12 溢出核对

- **按钮**：高度 32~36px，图标 14~18px，`nowrap`，超长省略
- **输入框**：`min-width: 0`，label 在上
- **chip**：高度 22~26px，超长截断，>3 个自动换行
- **表格**：min-width + `overflow-x-auto`，长文本 truncate + title
- **卡片**：不嵌套卡片（用分隔线），图片固定比例容器
